# AI Hackathon @ UC Berkeley 2026

## StoryStream

Turns any topic into an age-appropriate illustrated story + quiz via a 3-agent pipeline
(Researcher -> Storyteller -> Quiz Master).

- [`server/`](server/) - FastAPI backend, agent pipeline, Redis cache (see [server/README.md](server/README.md))
- [`frontend/`](frontend/) - React + Vite dashboard UI (see [frontend/README.md](frontend/README.md))
